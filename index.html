Prueba de Gramática - Nivel Intermedio

1. ¿Cuándo usamos el Present Simple?

  • a) Para acciones que ocurren en el momento.
  • b) Para hábitos y acciones regulares.
  • c) Para predicciones del futuro.

Respuesta correcta: b) Para hábitos y acciones regulares.

2. ¿Cuál es la diferencia principal entre "be going to" y "Present Continuous"?

  • a) Ambos son usados para predicciones.
  • b) "Be going to" se usa para decisiones ya tomadas, y "Present Continuous" para acuerdos futuros.
  • c) No hay diferencia, se usan de la misma manera.

Respuesta correcta: b) "Be going to" se usa para decisiones ya tomadas, y "Present Continuous" para acuerdos futuros.

3. ¿Cuál es la diferencia principal entre Present Perfect y Past Simple?

  • a) Present Perfect se usa para acciones que ocurrieron en un momento específico en el pasado.
  • b) Past Simple se usa para experiencias pasadas que no se pueden situar en un tiempo específico.
  • c) Present Perfect se usa para hablar de experiencias pasadas sin especificar cuándo ocurrieron.

Respuesta correcta: c) Present Perfect se usa para hablar de experiencias pasadas sin especificar cuándo ocurrieron.

4. ¿Cuándo usamos Present Perfect Continuous?

  • a) Para acciones que comenzaron en el pasado y continúan hasta el presente.
  • b) Para describir acciones que ocurrieron en el pasado pero que ya han terminado.
  • c) Para acciones pasadas que no tienen un resultado presente.

Respuesta correcta: a) Para acciones que comenzaron en el pasado y continúan hasta el presente.
